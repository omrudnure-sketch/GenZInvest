# GenZ Mutual Fund Adviser

The ultimate platform for GenZ investors. Live news, insights, and more.
Built with Next.js 14+, Tailwind CSS v4, and Framer Motion.

## Getting Started

First, install the dependencies (if you haven't already):

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Live News**: Real-time market updates with a GenZ aesthetic.
- **Dark Mode**: Fully immersive dark theme with neon accents.
- **Responsive**: Works on mobile, tablet, and desktop.

## Project Structure

- `app/`: Application routes and pages.
- `components/`: Reusable UI components (NewsCard, NewsFeed).
- `lib/`: Shared utilities and mock data.
- `hooks/`: Custom React hooks.
